Assistive Features

Assitance Prompts Use machine learning and heuristics to identify when a user may need help.

Object Detection Use computer vision to analyse the user's field of view and report detected objects and locations.

Fall Detection Use head tracking and machine learning to alert of potential falls.

Item Tracking Use spatial anchors to remember the location of significant places or items.

Sticky Notes Use spatial anchors to create virtual sticky notes.

Photo Diary Periodically take a picture automatically for the user to review if they wish.

Activity Suggestions Use machine learning and computer vision to identify the location of the user and provide context sensitive instructions e.g. steps to make a coffee.
